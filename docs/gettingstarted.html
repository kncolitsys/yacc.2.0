CMS Engine Installation Instructions

Requirements

  • ColdFusion
    • 8.0 or greater
    • Download
  • Tranfer Orm
    • v1.0 or greater
    • Download
  • Coldspring
    • 1.2 RC1 or greater
    • Download

Simple Install

  • Unzip yacc zip file
  • Install YACC engine
    • Copy yacc folder in webroot
  • Setup Database and datasource
    • Find appropriate sql install file in sql folder
    • Create datasource in ColdFusion administrator named yacc
  • Edit ColdSpring Config File
    • Open file"/yacc/config/coldspring.xml.cfm"
    • Set values according to your site's needs.
  • Install YACC site
    • Copy yacctemplate folder to webroot
    • Rename folder to whatever you want
    • Open "/yacctemplate/config/coldspring.xml.cfm"
    • Set values according to your site's needs.
  • Navigate to your YACC site and enjoy

What are the various pieces of the CMS?

Here are a few definitions of the terms used throughout the documentation:

Page
A page is a page, no surprises here. Pages can contain sections and nuggets, they are modified by layouts.
Nugget
A nugget is a section of code that can appear on multiple pages. It displays exactly the same on every page. It has an ID. There can be only one nugget with the same ID per installation of YACC. There can be only one!
Section
A section is a section of code that is unique to the page on which it appears. It has an ID. There can be only ID per page, but there can be multiple section on different pages with the same ID. There can be more than one!
ID
A unique identifier for the nugget on the site, or the section on the page. Sections and nuggets are surrounded by divs with an ID of their ID.
Layout
Layouts are versions of the same template. For example you could have a two column and a three column version of the same layout, with nuggets in different spots.
Template
Template refers to the overall layout, color scheme and typography of the site. There are three templates installed by default with YACC.
